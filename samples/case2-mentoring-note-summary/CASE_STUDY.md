# 멘토링 일지 위험 신호 요약 에이전트

## 사례 메타데이터
- 사례 ID: impact-agent-20260605-mentoring-risk-summary
- 공개 범위: 내부 공유만
- 분야: 교육
- 재사용 수준: 분야 특화

## 문제 요약
청소년 멘토링 일지에서 긴급 확인이 필요한 신호를 사람이 늦게 발견한다.

## 기존 방식
코디네이터가 모든 일지를 읽고 확인 필요 항목을 표시한다.

## AI 에이전트 워크플로
익명화 일지 입력 -> AI 요약 -> 코디네이터 우선 검토 -> 담당자 공유

## 사람 검토
담당자 공유 전 코디네이터 검토

## 사용 도구
Codex, ChatGPT

## MVP 결과물
확인 필요 우선순위표, 위험 신호 근거, 후속 질문, 코디네이터 공유 메시지

## 성공 기준
확인 필요 일지가 표 상단에 오고 각 일지마다 근거 문장과 사람이 확인할 질문이 1개 이상 나온다.

## 배운 점
민감한 청소년 사례는 공개 사례 작성 시 업무 흐름만 남기고 원문과 신호 예시는 익명화해야 한다.

## 재사용 가이드
위험 신호 기준과 후속 질문 문구를 프로그램별로 바꾼다.

## 익명화와 공개 제외
이름, 학교, 연락처 등 식별 정보를 제거한다.
