# 생활 불편 제보 분류 에이전트

## 사례 메타데이터
- 사례 ID: impact-agent-20260605-neighborhood-repair-router
- 공개 범위: 공개 가능
- 분야: 지역
- 재사용 수준: 그대로 재사용 가능

## 문제 요약
주민 제보가 자유형식으로 들어와 담당 분야 분류와 답변 초안 작성이 늦어진다.

## 기존 방식
활동가가 제보를 읽고 분류한 뒤 답변을 직접 작성한다.

## AI 에이전트 워크플로
제보 텍스트 입력 -> AI 분류/답변 초안 -> 활동가 검토 -> 주민 답변

## 사람 검토
주민에게 답변하기 전 활동가 검토

## 사용 도구
Codex, ChatGPT

## MVP 결과물
제보 분류표, 담당 후보, 위치 확인 질문, 주민 답변 초안

## 성공 기준
각 제보가 하나의 유형으로 분류되고 담당 후보와 주민에게 보낼 다음 답변이 빠짐없이 나온다.

## 배운 점
텍스트 분류와 답변 초안은 3시간 MVP에 적합하지만 실제 접수 자동화는 이후 단계로 두는 편이 안전하다.

## 재사용 가이드
지역별 담당 후보 목록과 답변 톤을 바꾼다.

## 익명화와 공개 제외
연락처와 상세 주소를 제거한다.
