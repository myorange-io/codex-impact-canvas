# 긴급식품지원 신청 검토 보조 에이전트

## 사례 메타데이터
- 사례 ID: impact-agent-20260605-case-dec63a0f
- 공개 범위: 익명화 후 공개 가능
- 분야: 복지
- 재사용 수준: 템플릿 재사용 가능

## 문제 요약
긴급식품지원 신청서와 상담 메모를 사람이 반복해서 읽고 지원 우선순위를 정리한다.

## 기존 방식
담당자가 신청서와 상담 메모를 읽고 스프레드시트에 긴급도를 적는다.

## AI 에이전트 워크플로
익명화 텍스트 입력 -> AI 초안 생성 -> 담당자 검토 -> 회의 자료 반영

## 사람 검토
지원 여부 확정 전 담당자 검토

## 사용 도구
Codex, ChatGPT

## MVP 결과물
긴급도 우선순위표, 건별 근거 요약, 담당자 확인 질문, 검토 체크리스트

## 성공 기준
샘플 3건 모두 긴급도와 근거가 표시되고 담당자가 회의에서 바로 확인할 질문을 볼 수 있다.

## 배운 점
민감한 복지 판단은 자동 확정보다 담당자 검토용 초안에 집중해야 안전하다.

## 재사용 가이드
긴급도 기준과 확인 질문을 기관별로 바꾼다.

## 익명화와 공개 제외
이름, 연락처, 주소를 제거하고 가구 특성만 남긴다.
